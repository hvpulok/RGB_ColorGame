Pulok's Color Guessing Game
Guess Color Combination based on :
RGB(255, 255, 255)

Instructions to Play the Game:

  • RGB means Red, Green, Blue. Each value ranges from 0 to 256.
  • If R = 256, G = 0, B = 0, the combination will be Red color
  • If R = 0, G = 256, B = 0, the combination will be Green color
  • If R = 0, G = 0, B = 256, the combination will be Blue color
  • Now if R= 256, G=100, B=1, the combination will be Redish+Greenish = yellowish color
  • Look at the RGB figure to guess approximate color and click the square
